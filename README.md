# SPGP_2024

# 게임 컨셉
플레이어가 캐릭터를 조작하여(점프와 슬라이드) 장애물들을 피해 주어진 시간 안에 최대한 많은 젤리를 먹는 러닝액션 게임

# 개발 범위
#### 1. 캐릭터가 앞으로 계속 움직이는 효과를 주기위해 배경이 스크롤 되는 것을 구현
#### 2. 캐릭터의 점프와 슬라이드 모션과 기능 구현
#### 3. 젤리와 캐릭터, 캐릭터와 장애물간의 충돌처리
#### 4. 플레이어가 획득한 젤리의 점수를 화면에 표시
